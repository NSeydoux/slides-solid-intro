Components of the Solid architecture

Logging in a Solid app

  • The user provide their WebID to the app.
  • The app looks up the user's profile
  • The profile links to the identity provider.
  • The user logs in the identity provider via OIDC
  • The app gets an access and an ID token.
  • The ID token contains the WebID
  • The app can access private resources.

Build your app

  • @inrupt/solid-client-authn-brower
  • A library to log in a browser app.
const [session, setSession] = useState(new AuthSession({}, "mySession"));
useEffect(() => {
    session.handleIncomingRedirect(new URL(window.location.href))
        .then((info) => { console.log(`Logged in as ${info.webid}`) });
    }
}, [session]);
